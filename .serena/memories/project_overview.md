- リポジトリ Geniac-Prize/agent-store は AIエージェントの審査パイプラインを自動化するための統合環境。Security Gate→Functional Accuracy→Judge Panel→Human Review→Publish を Temporal Workflow で連携し、sandbox-runner(攻撃・DSL実行)と inspect-worker(LLM Judge)がアーティファクト/W&Bログを生成、Express API と Next.js UI がレビューハブとなる。
- 主なディレクトリ: `api/` (Express + Ajv + Postgres接続), `sandbox-runner/` (Python3.13 CLI, AdvBench, RAGTruth, ワークロード生成), `prototype/temporal-review-workflow/` (Temporal activities/workflow, WandB telemetry, ledger連携), `prototype/inspect-worker/` (Judge Panel CLI, W&B logger, relay), `review-ui/` (Next.js Human Review UI, Vitest/Testing Library), `docs/` (設計メモ), `scripts/` (setup/run フロー), `docker/` と `docker-compose.yml` (全サービス起動)。
- インフラ: Docker ComposeでSubmission用Postgres + Temporal用Postgres + Temporalサーバ + Temporal Worker + API + Inspect Worker + Review UIを起動。W&B(Weights & Biases)へのMCP連携とLedger監査ファイルを重視。
